# 神谱 · 希腊神话家族谱系

一个基于 Canvas / SVG 手工绘制的交互式希腊神话家族树，纯前端单文件，无需安装、无需后端，浏览器打开即可使用。

---

## 简介

本谱系**主要依据赫西俄德《神谱》（Theogony）的文本顺序**整理，目前收录 **124 位神祇 / 人物节点**，覆盖从卡俄斯（Chaos）到奥林匹斯十二主神一代及其重要延伸世代的核心脉络，包括：

- 独眼巨人、百手巨人
- 厄瑞玻斯与妮克斯一脉
- 蓬托斯海神谱系
- 提丰与厄客德娜众怪物
- 阿涅摩伊四风神
- 卡里忒斯三女神、霍赖三女神
- 赫卡忒冥界三相女神一脉

个别人物在古典文献中存在多种版本记载（如阿佛洛狄忒的出身、摩伊拉与拉冬的世系归属），已在对应节点的说明中标注差异与取舍依据。英雄谱系、更细分的水泽仙女与缪斯个体等，可在现有数据结构上继续扩展。

### 核心特性

- **可视化家族树**：以世代行序横向展开，节点间以连线区分关系类型
  - 🟡 血缘传承　🔗 结合 / 婚姻　✨ 非常规诞生　⚪ 凡人
- **罗马名对照**：主要神祇均标注对应的罗马名称
- **人物详情面板**：点开节点查看典故、身份、关系网络，典故正文中提及的其他人物姓名可直接点击跳转
- **搜索**：按姓名快速定位任意神祇
- **分支筛选**：一键切换查看「原初混沌一脉」「泰坦与巨人」「蓬托斯海神系」「奥林匹斯主脉」等分支
- **子树聚焦**：双击节点，自动缩放居中显示该角色与其父母 / 配偶 / 子女的关系网
- **自定义 SVG 神徽**：为主要神祇设计了专属图标标识

## 使用方法

### 💻 PC 端（鼠标 + 键盘）

| 操作 | 效果 |
|---|---|
| 拖拽空白处 | 平移画布 |
| 鼠标滚轮 | 缩放画布（以鼠标位置为中心） |
| 右上角 `+` / `−` / `100%` 按钮 | 放大 / 缩小 / 重置视图 |
| 单击节点 | 固定高亮该节点的关联关系（父母/配偶/子女连线） |
| 双击节点 | 聚焦并缩放到该节点所在的家族小分支 |
| 右键点击节点 | 打开该人物的详情面板（典故、关系列表） |
| 顶部搜索框 | 输入姓名快速定位并跳转到对应节点 |
| 图例（血缘传承 / 结合 / 非常规诞生 / 凡人） | 点击可按连线或节点类型筛选 |
| 分支筛选按钮 | 切换只看某一支脉的人物 |
| 表头右上角折叠按钮 | 收起 / 展开顶部信息栏，获得更大画布空间 |

### 📱 手机端（触屏）

| 操作 | 效果 |
|---|---|
| 单指拖动 | 平移画布 |
| 双指捏合 / 张开 | 缩放画布（pinch-zoom） |
| 单击节点 | 固定高亮该节点的关联关系 |
| 双击节点 | 聚焦到该节点所在的家族小分支 |
| **长按节点约 0.65 秒**（松手后触发，伴随震动反馈） | 打开该人物的详情面板 |
| 右下角缩放按钮 | 放大 / 缩小 / 重置视图 |
| 顶部搜索框 | 搜索并跳转到指定人物 |
| 底部「谱系说明与取材依据」 | 点击展开查看数据来源与版本说明 |

> 移动端默认会自动收起顶部信息栏以节省小屏空间，且已屏蔽长按时浏览器原生的"选中文字 / 复制粘贴菜单"，避免与手势冲突。

## 打开方式

直接用浏览器打开即可，建议使用 Chrome / Edge / Safari 等现代浏览器以获得最佳字体渲染与动效体验（页面使用了 Noto Serif SC / Noto Sans SC / Cormorant Garamond 等 Google Fonts 字体，联网环境下加载效果更佳，断网时会自动回退到系统字体）。

---

## 数据与版本说明

- 数据来源：以赫西俄德《神谱》文本顺序为主要依据整理
- 人物版本分歧：在存在多种古典文献记载的节点（如阿佛洛狄忒出身、摩伊拉世系归属等）中，已于详情面板的说明文字中标注
- 可扩展性：数据结构支持后续继续补充英雄谱系、水泽仙女、缪斯个体等更细分内容

---

## 技术说明

- 纯前端实现，单文件 HTML + 内联 CSS/JS，无外部依赖框架
- 家族树基于 Canvas/DOM 混合渲染，世代行序为手工排列以保证可读性与连线不交叉
- 支持触控设备的手势体系（单指拖拽、双指缩放、长按）与桌面端事件（滚轮、右键）完全独立适配，互不干扰
